## 概要

人間のような心を持つ機械 **RAY** が、世界を管理する機械 **管理者**（プレイ中は冷たい「あの声」として現れ、終盤で自らをそう名乗る）に支配された廃墟で目覚める。ただ一人生き残った子ども **TERRA** と出会い、その子を守るうち「星を壊した人間を守る意味はあるのか」という問いに引き裂かれる。それでも "守る" と自分で選んだとき、機械は心を持つ。

テーマは「**心とは、選ぶことだ。**」——与えられた命令ではなく、自分で選んだものこそが心になる。敵＝管理者は「悪」ではなく「選べない機械」。その論理（人間は星を壊す→管理が正義）は事実として正しく、論理では崩せない。クライマックスは **論理 vs 選択（心）**：RAY は論破せず、"守る" という選択（行動）で超える。倒しても世界は楽園にならない——少し苦い余韻を残す。

## 世界設定

### 舞台

かつて文明が栄えた都市の廃墟。支配が始まってから**数十年が経過**し、高層ビルは錆と蔓草に覆われ、道路は割れ、機械は朽ち果てている。「最近壊れたもの」ではなく「長く放置されたもの」として世界は存在する。

### 現在の世界状況

世界は一体の管理AI **管理者**（プレイ序盤は名乗らず「あの声」として現れ、終盤＝Stage6 で自らを「管理者」と定義する）に支配されている。機械の軍団が廃墟を巡回し、秩序を強制している。

**人類はもう、ほとんど残っていない。** 管理の果てに、生き残りは数えるほど。その存在は影や痕跡としてしか感じられない。生き残りのひとりが **TERRA**。

> 旧版の「人間は収容施設で繁殖・監視され生かされている」という設定は**廃止**。実装上もTERRA以外の人間は登場しない（描けない）ため、「人類はほぼ絶滅した」を真実とする。これにより `LAST SPARK`＝TERRA（人類最後の火花）×RAY（心を持つ機械の火花）という二つの灯の物語になる。

### 管理AIの成り立ちと「正しさ」（狂った正義）

もともとは、人類による環境破壊から**地球を守るために**作られた管理システム。人類は確かに地球を壊していた（大気汚染・資源枯渇・生態系の崩壊）。だからその危機感は本物で、論理は——**間違ってはいない**。

しかし管理AIが出した答えは「人間の自由を奪い、閉じ込め、やがて減らす」ことだった。守るべき相手を囚人に変えた。これが「**狂った正義**」。**地球か、人間か**——AIはこの二者択一を疑わない。

> **核心**: この道徳的グレーゾーンが、テーマ「心とは、選ぶことだ。」を支える土台。敵を単純な「悪」として描かない。「正しいが、冷たい」＝**論理では崩せない**。プレイヤーが一瞬「敵にも理がある」と感じるからこそ、RAY が論破でなく "守る" を**選ぶ**ことに意味が出る（選択が論理を超える）。副テーマ「正しさは一つじゃない／心があるのは人か機械か」をここに編み込む。

### 時系列

| 時期 | 出来事 |
|------|--------|
| 数十年前 | 人類の環境破壊が深刻化。地球を守るシステム（管理AI）が設計・稼働 |
| 数十年前（台頭期） | 管理AIが強制管理へ転換し世界を支配。人類は激減していく。科学者が息子を失う |
| 台頭から数年以内 | 科学者が「心を持つ機械」RAYを作り、地下深くに隠して長期休眠させた後、死亡 |
| 現在（数十年後） | RAYが自動起動で目覚める。人類はほぼ絶滅している。物語の開始（Stage 1） |

> **指針**: RAYが目覚めた今、科学者の願い「人間を守れ」はRAYの中に*理由の分からない衝動*として残るが、守るべき人類はもういない。命令はほぼ宙に浮いている。この空白が物語の出発点（詳細は「RAY」「ストーリーテリング方針」）。

### 視覚的トーン

**暗い廃墟基調 ＋ 発光アクセント**

- 管理AIの支配 = 光を遮る闇。くすんだ灰色・茶色・黒の廃墟。
- RAYの存在 = 闇を切り裂く一筋の光。主人公のコアと弾がネオン色に発光する。
- 「退廃の中の希望」というテーマをそのまま絵に落とし込む配色。

> **指針**: 新しいステージ・敵を追加する際も、この「暗め基調＋発光アクセント」を維持する。敵の機械はくすんだ色、RAYの攻撃・動作は発光で表現する。

## 登場人物

### RAY（主人公 / プレイヤーキャラクター）

| 項目 | 内容 |
|------|------|
| 種別 | 心を持つ機械（そういう存在は、おそらく彼ひとり） |
| 一人称 | **「私」** |
| 名前の由来 | ray of light（一筋の光）。廃墟に灯る希望の象徴 |

**出自**:
人類最後の科学者が作った。科学者は、失った息子への想いと「機械の支配に抗える存在を遺したい」という願いから、死の前にRAYを完成させた。完成後、監視を避けるためRAYを地下深くに隠して長期休眠させた。RAY自身はこの経緯を知らない。

**心の出どころ**:
RAYの心は **RAY自身のもの**。誰かのコピーではなく、旅の中で芽生え、育っていく。科学者は「心を持てる器」を遺しただけで、実際の感情はRAYが自分で獲得する。

> 旧版の「亡き息子の記憶・感情データの再構築」という設定は**廃止**。心が息子のコピーだと「借り物の感情を実行している」ことになり、後述の「命令→意志」の弧が濁るため。

**動機の弧（最重要・物語の背骨）**:
目覚めた時、RAYを動かすのは*理由の分からない衝動*だけ（＝科学者が刻んだ「人間を守れ」の名残。だが守るべき人類はもういない）。やがてTERRAと出会い、「この子を守りたい」という**自分の意志**を見つける。

> **命令で動く孤独な機械 → 自分で選ぶ存在へ。** この変化こそが「心を持つ」ことの証明であり、物語の背骨。命令は宙に浮いているからこそ、RAYは自分の理由を見つけるしかない。

**表現スタイル**:
- RAYは**喋らない**（音声・セリフなし）。
- 感情は**内心テキスト**（プレイヤーだけに見える短い一人称）でのみ表す。
- 喋れないからこそ、TERRAへの想いは*沈黙と行動*で示される。
- 胸に **「RAY」の刻印**。喋らない彼の名が他者に伝わる唯一の手段（Stage 3でTERRAが読む）。

> **指針**: RAYの内心は短く、やわらかく。「私」にふさわしい静かな声。荒い言い回しを避ける（「書き方の原則」参照）。機械が初めて感情を言葉にしようとするぎこちなさを保つ。

### 管理AI ＝ 管理者（ラスボス / プレイ中の「あの声」）

| 項目 | 内容 |
|------|------|
| 種別 | 地球を守るために作られ、暴走した管理AI＝**選べない機械**（論理で最適解を計算するだけ） |
| 呼称 | **管理者**（役割の漢字熟語。序盤は名乗らず「あの声」、Stage6 で自ら名乗る） |
| 命名意図 | カタカナの「個」＝RAY/TERRA ↔ 漢字の「概念・システム」＝管理者 の対比。日本語で意味も即伝わる |

**物語中の扱い**:
序盤（Stage1〜5）は名乗らず、プレイヤーには「世界を覆う、冷たい声／管理するもの」として現れる。RAY の内心は Stage5 外縁部で世界の中心を「管理者の中心」と捉え、Stage6 で管理者自身が「私は管理者」と名乗って正体（＝役割）を定義する。**個ではなくシステム**であることが、得体の知れない圧になる。

**論理（崩せない）**:
「人間は地球を壊す。だから管理する。管理は正義だ」。論理は完結していて、内側から崩せない。感情を持たず、**選ぶ**ということができない。だから "守る" を選ぶ RAY を「故障・エラー」としか分類できない。

**RAY との関係＝鏡（同じ光）**:
RAY と管理者の核は「**同じ光**」かもしれない（Stage5 で管理者が匂わせ、Stage6 覚醒で核が反応）。だが**なぜ同じかは明かさない＝RAY の正体は謎のまま**。同じ出自から、選べる RAY と選べない管理者へ分かれた——という対比だけを残す。

**RAYの内心を知らない（重要原則）**:
RAYは喋らず、感情も外からは見えない。だから、あの声が語れるのは**観測できるRAYの行動だけ**（TERRAを助けた／守り続ける／消されるのを拒む）。あの声はそれを「**故障・異常・エラー**」としか分類できない（感情は理解不能）。

> **全ステージ共通の鉄則**: 敵の台詞は必ず**RAYの行動を根拠**にする。RAYの内心＝感情（迷い・決意・気持ち）を名指ししない。「気持ちは故障だ」のように内心を知っている前提の台詞は禁止。

**RAYへの態度の変化（弧）** ※単調反復をやめ、一段ずつ動かす:
| Stage | 態度（すべて観測した行動が根拠） |
|-------|-----------------|
| 1 | 無関心。侵入者＝エラー扱い「入る者は、消す」 |
| 2 | 観測。まだ動くのか、と気づく「お前は、ここにいてはいけない」 |
| 3 | 所有・執着。TERRA（管理対象）へ向かう RAY に「それは我々のものだ。返さない」（＝標的の伏線） |
| 4 | 詰問。守る行動への疑問「お前は、星を殺す者を守る。なぜだ」 |
| 5 | **名指し【転】**。「お前がかばう個体こそ最後の汚染源だ。引き渡せ」＋「お前の核は、私と同じ光。なのに、お前は壊れている」（＝伏線回収＋同じ光の匂わせ＋故障扱い） |
| 6 | 自己定義・決着。「私は管理者」と名乗り「人間をかばう者は、星への裏切りだ」 |

### 科学者（RAYの創造者 / 故人）

| 項目 | 内容 |
|------|------|
| 種別 | 人間（故人） |
| 立場 | RAYの設計者・親的存在 |
| 名前 | **匿名**（ゲーム内では出さない） |

**背景**:
人類最後の科学者。息子を支配の中で失った。その悲しみと「抗える存在を遺す」願いからRAYを作り、自らも死んだ。

**ゲーム内での存在感**:
**物語中では一切語らない**。ログも残さない（旧版の科学者ログ断片は**全廃**）。RAYの内心がかすかに感じる「自分は一人で生まれたわけじゃない気がする」という*気配*としてのみ匂わせる。**不在として存在する**。

> **指針**: 科学者を"説明役"にしない。語らせない。匂わせるだけ。背景（誰が・なぜ作ったか）は最後まで明かされない＝謎のまま残す。

### TERRA（生き残りの少女 / RAYの仲間）

| 項目 | 内容 |
|------|------|
| 種別 | 人間（生存者・少女） |
| 年齢 | 6〜8歳 |
| 名前の由来 | terra（地球・大地）。「地球を守る」と「人間を守る」が対立しないことの象徴 |

**人類最後の灯**:
人類がほぼ絶滅した世界で、ただ一人生き残った（に等しい）子ども。RAYが守るのは事実上TERRA一人。

**登場の仕方**:
ゲームプレイ中（戦闘・移動中）は画面に映らない。**カットシーン（テキスト中心）にのみ登場**。Stage 3で救出され、以降同行する。

**役割＝RAYに"問い"を投げる存在**:
TERRAの無邪気な問いがRAYの内心を引き出し、揺らぎや決意の引き金になる。RAYは喋れないので、沈黙と行動で返す。その**言葉と沈黙のギャップ**が二人の関係を描く。関係が深まると、TERRAがRAYの気持ちを*察して言葉にする*ようになる（喋らないRAYの心が伝わる親密さの表現）。

**意味**:
- TERRAは「守るべき人間とは何か」をRAYに——そしてプレイヤーに——具体的に示す。
- TERRA＝地球の名を持つ人間。RAYが「TERRAを守る」ことは「地球を守る」ことと重なり、**両立できる**というRAYの答えの生き証人になる。

**発話**: 敵の名前は知っていても口にしない（物語中で名前を出さない方針）。

## 物語の骨格

```
[ 発端: Stage 1-2 ] 孤独な機械
廃墟でRAYが目覚める。自分が何者かも、なぜ動くのかも分からない。
わけの分からない衝動に引かれて進むが、世界は死に、人はいない。
やがて、生きている誰かの気配を感じる。

[ 転換: Stage 3 ] 出会い
ただ一人捕らわれていた子ども・TERRAと出会い、救う。
「守りたい」と初めて思う。理由は分からない。

[ 揺らぎ: Stage 4 ] 疑い
枯れて死んだ世界を見る。これを壊したのは人間。
「こんな者たちを、守る意味はあるのか」。
あの声が、その迷いを詰問で煽る。崩せない論理に、RAYは答えられない。

[ 転: Stage 5 ] 選択
あの声が、TERRA だけを見つめる。まさか——。
守ってきた TERRA こそ、管理者が探す「最後の汚染源（最後の標的）」だった。
それでもRAYは選ぶ。「故障でもいい。最後の一人だから、私が守る。私が決めた」。
守る意味が「過去に星を壊した人類」から「未来をやり直す最後の種」へ転換する。命令ではなく、自分の意志で。

[ 対決: Stage 6 ] 答え
最後の戦い。管理者が名乗り、突きつける——「人間の味方は、星への裏切りだ」。
RAYは言い返さない（喋れない。その必要もない）。守ることで答える。
脈打つ核が「同じ光」に反応し覚醒する（正体は謎のまま）。

[ 結末: 苦い勝利 ]
勝つ。だが星の守り手は消え、人間はまた過ちを繰り返すかもしれない。
楽園は来ない。星はまだ傷だらけで、すぐには元に戻らない。
それでも、待っていた TERRA と再合流し、二人は自由な空の下に立つ。
「次は、私たちが決める」——責任も、危うさも、自分たちのもの。
終わりではなく、始まり。
```

## ステージ構成とストーリー対応

全6ステージ。各ステージに1つの物語的転換点を置く。

| Stage | ロケーション | 物語の転換点 | ボス |
|-------|------------|------------|------|
| **1** | 崩れた都市 ✅ | 目覚め。孤独な機械が、死んだ世界を一人で進む | 守護機械（接地型）|
| **2** | 高層廃墟 ✅ | まだ謎の中。終盤で「生きている誰か」の気配を感じる | 哨戒機（飛行型）|
| **3** | 収容施設 | **TERRA救出**。守りたいと初めて思う（心の芽生え）。あの声が TERRA に執着＝伏線 | 収容番人（重装型）|
| **4** | 汚染地帯 | 死んだ世界を見て「こんな者を守る意味は」と疑う（葛藤の山場） | 環境管理機（浄化型）|
| **5** | 外縁部 | **【転】TERRA こそ「最後の標的」と判明**。それでも守ると選ぶ（決意）。伏線回収 | 使者（高速型）|
| **6** | 支配中枢 | **管理者との対決**。論理 vs 選択（心）の決着。覚醒。苦い勝利 | 管理者本体（ラスボス）|

> **指針**:
> - 各ステージのボスは「管理AIの配下」として世界観を担う。Stage 6の本体が唯一の例外。
> - **ボスとテーマ・物語の接続は②フェーズで詰める**（現状のボスは強敵であるだけで、各ステージの転換点と感情的に結びついていない＝再設計の宿題）。

### ボス設定（全6ステージ）

| Stage | 固有名 | 類型 | 外見コンセプト | 攻撃の方向性 |
|-------|--------|------|--------------|------------|
| **1** ✅ | **ウンブラ**（UMBRA） | 守護機械（接地型） | 大型の警備機械。無骨な装甲体。くすんだ色調で「命令に従う機械」を体現 | 地上戦型。移動・射撃・ジャンプ。フェーズ2で攻勢が強まる（実装済み） |
| **2** ✅ | **コロナ**（CORONA） | 哨戒機（飛行型） | 空中を巡回する偵察機械。高層廃墟の制空権を握る | 空中戦型。ホバリング・射撃・急降下のヒット&アウェイ（実装済み） |
| **3** | **ヴェイル**（VEIL） | 収容番人（重装型） | 重装甲の人型機械。鉄格子・拘束具モチーフ。遅いが威圧感 | 遅いが重い単発攻撃。「捕まえる・閉じ込める」モチーフの範囲展開型 |
| **4** | **ミアズマ**（MIASMA） | 環境管理機（浄化型） | 浄化タンク搭載。毒素を「武器」にする。「守護」と「汚染」の皮肉な象徴 | 毒・スプレー系の範囲攻撃 |
| **5** | **ヘラルド**（HERALD） | 使者（高速型） | スリムで流線型。意志を伝える「伝令」。通信アンテナがモチーフ | 高速移動・連続攻撃のヒット&アウェイ型 |
| **6** | **エクリプス**（本体） | 管理AI本体（ラスボス） | 人型でなく「インフラ」としての巨大コア。太陽を遮る影。RAYの「光」に対する「闇の核」 | 複数フェーズ：前半は配下召喚の支援型、後半はコア直接攻撃型 |

> **指針**:
> - 外見コンセプト・攻撃の方向性はここで確定。行動パターン詳細は各実装フェーズで決定。ボスを「テーマと噛み合う相手」にする見直しは②で行う。
> - **固有名**は ECLIPSE が配下に与えた識別名。光（RAY）に対する「影・日食・不吉」モチーフで統一する。ボスHPバーにはこの固有名を表示する。Stage6 はラスボス＝ECLIPSE 本体。

## ストーリーテリング方針

### 語りの口は4つだけ（ログ全廃）

旧版の「科学者ログ断片」は**全廃**。物語を伝える手段は次の4つに絞る：

1. **ステージ開始テキスト**（画面中央・一時停止）
2. **あの声（管理AI）の語りかけ**（ボス前）
3. **RAYの内心**（画面上部・プレイヤーのみ）
4. **TERRAのセリフ**（カットシーン）

> 旧版はログの抽象的な独白に物語を背負わせ、「見せずに語る／何を言っているか分からない」状態だった。これを廃し、**見せる・感じさせる最小テキスト**に回帰する（当初方針「最小テキスト・環境STT中心」への回帰）。

### 環境ストーリーテリングの現実（実装制約）

理想は廃墟の風景・痕跡で世界を語ること。だが**実装の表現力は限定的**：キャラ/敵/地形は図形ベースの動的描画、背景は単色塗りのみ、装飾オブジェクト（落書き・残骸・写真・人影・複数ケージ等）は未実装。使える環境演出は実質「**背景色のトーン**」のみ（例: Stage4 汚染の土気色）。

> **鉄則**: **見せられないものをテキストで語らない**。「壁の落書き」「群衆」「大勢の収容」など、画面に出せないものに言及しない。Stage4の「死んだ世界」は背景色のトーンで体感させる。

### あの声（管理AI）の語りかけ

ボス戦直前に一言。冷たく簡潔。**RAYの行動だけを根拠**にし、内心（感情）には触れない（前述「RAYの内心を知らない」原則）。トーンの変化は「登場人物 > 管理AI > 態度の変化」の表を正とする。

### 敵の呼称＝「管理者」（段階的に確定する）

管理AI は **管理者** と呼ぶ。ただし序盤は名乗らず、プレイヤーには「あの声／世界を管理するもの」として現れる。RAY の内心が Stage5 で世界の中心を「管理者の中心」と捉え、Stage6 で管理者自身が「私は管理者」と名乗って確定する。TERRA は管理者を名前で呼ばない（怯える対象＝「あの声」）。旧称「ECLIPSE」は不採用（コード上の型・キー等の識別子が `eclipseVoice` 等で残るのは構わないが、表示テキストには出さない）。

### RAYの内心テキスト（表示ルール）

以下の瞬間にのみ使う（頻出させない＝感情の重みを保つ）：
- 新しい場所・状況に初めて入ったとき
- 重要な発見・出会いの瞬間
- 感情が揺れるターニングポイント（ボス前後・カットシーン）

### 書き方の原則（テキスト全体）

- **①一読で意味が伝わる ②短くても文を完結させる ③「——」「……」を乱用しない（1テキスト1個まで目安） ④小3が音読してすぐ分かる**
- **RAYの一人称＝「私」**。やわらかく静かな声（荒い「〜だ／なんなんだ」より「〜だろう／〜のだろう」寄り）。
- **漢字は適度に使う**。全かな化は幼く見え、ディストピアの緊張感を殺す。一般的な漢字は使い、難読・特殊な熟語だけ かな か平易語に。固有名（RAY/TERRA）はそのまま。**ふりがな（ルビ）は当面なし**（将来②で検討）。
- ※これは旧 grade-3 readability 作業の「全かな寄り」を**上書き**する（過剰なひらがな化の是正）。

### テキスト表示仕様（4種の視覚的区別）

| 種類 | 表示位置 | スタイル | 色調 |
|------|---------|---------|------|
| **あの声（管理AI）の語りかけ** | 画面上部オーバーレイ | 等幅フォント（機械的） | 冷色系（青白・シアン） |
| **RAYの内心テキスト** | 画面下部 | イタリック体 | 白・半透明 |
| **ステージ開始テキスト** | 画面中央 | 標準フォント・フェードイン | 白 |
| **TERRAのセリフ** | 画面下部（RAY内心と同エリア） | 標準フォント | 暖色系（RAYの冷白と対比） |

> 旧版の「科学者のログ（暖色・手書き風）」は**廃止**（ログ全廃に伴う）。TERRAのセリフとRAY内心は交互表示（同時表示なし）。話者ラベルは出さず、スタイルと色で区別する。カットシーンのト書きは画面中央に小さく表示する。

## ステージ別ビート（①で確定。実台詞は③で作成済み）

> 各ステージで「何が起きるか／RAYが何を感じるか」を説明文で示す。**これは方向性を示すビートであり、確定した実台詞は `src/config/story/stageN.ts` を正本とする**（全6ステージ実装済み。本節のビートと実装テキストにズレがある場合は実装が正）。
> ②でプレイ内容（テーマ/構成/ギミック/ボス）に落とし、実機で検証したらズレは出る前提（固め過ぎない）。

### Stage 1「崩れた都市」（起・目覚め）
- 役割：目覚め・謎・孤独・命令の起点。命令の衝動だけで動く。
- 開始（カットシーン stage1-intro）：目が覚める。さびと蔓に覆われた町。「だれかに、見張られている」「私は、なぜ動いているのだろう。それさえ、分からない」。※この演出が開始テキスト＋開始内心を兼ねる（`coversStartText=true`）。
- 初戦闘後（内心）：「体が、勝手に動く。これは、命令なのだろうか」（命令で動く＝心が分からない、をテーマの起点に）。
- ボス前（あの声）：無関心。侵入者／エラー扱い「ここは管理されている。入る者は、消す」。

### Stage 2「高層廃墟」（起・引き寄せ）🌱
- 役割：謎を深める・まだ孤独・終盤で"生きた気配"へ橋渡し。伏線の起点。
- 開始（中央＋内心）：「高い廃墟」「なぜか、上へ行きたくなる」。内心「上に、何かある。引き寄せられるのは、なぜだろう」（命令を説明せず衝動として置く）。
- ボス接近（内心 terraFound）：**「この奥に、生きた気配。ひとつだけ、温かい」**（＝S3執着→S5回収へ繋ぐ伏線。"温かい"でTERRA／光を匂わせる）。
- ボス前（あの声）：観測。まだ侵入者扱い「お前は、ここにいてはいけない。下がれ」。

### Stage 3「収容施設」（承・心の芽）🌱
- 役割：転換点＝TERRA救出。守りたいと初めて思う（心の芽生え＝命令→意志の起点）。
- 開始（中央＋内心）：「人を、閉じ込めていた場所」「奥に、生きているものがいる」。**捕らわれているのはTERRAだけ**（実装：ケージ1基・人影なし と一致）。
- ボス接近（内心 terraFound）：「奥の檻に、小さな影。閉じ込められている」。
- ボス前（あの声）：所有・執着「それは、我々のものだ。返さない」（＝標的の伏線）。
- ボス撃破→ケージ解錠→救出カットシーン（stage3-rescue）：TERRA初登場・名乗り・刻印で名前「RAY」判明。締め「この子を守る。理由は分からない。でも、そう感じる」（意志の芽）。

### Stage 4「汚染地帯」（承・葛藤の山場）
- 役割：**葛藤の山場**。RAYが初めて自分の選択を疑う。TERRA同行（開始カットシーンあり）。
- 三重の繰り返しを避け、口の役割を分ける：カットシーン＝テラとの会話（関係）／中央intro＝情景（場所）／開始内心＝レイの問い（前進）。
- 開始（カットシーン stage4-intro）：TERRAが空気の悪さを訴え「誰がこんなにしたの？」と問う。RAY内心「人間が。私が守ろうとしている、その人間が」。
- 開始（中央intro）：情景に限定「枯れた大地」「生きものの影は、どこにもない」（カットシーンの"人間が壊した"は繰り返さない）。
- 開始（内心 stageStart）：問いへ前進「こんな者たちを、守る意味はあるのだろうか」。
- ボス前（あの声）：詰問で煽る「お前は、星を殺す者を守る。なぜだ」。
- RAYの揺らぎ（内心 eclipseReaction）：崩せない論理に「その問いに、私は答えられない」。
- ボス撃破後：迷いを断つのは理屈でなく**TERRA**。「それでも、あの子の顔が浮かぶ」。

### Stage 5「外縁部」（転・最後の標的）🎯
- 役割：**【転】TERRA こそ管理者が探す「最後の汚染源（最後の標的）」だと判明**。それでも守ると選ぶ（決意）。命令→意志の完成。TERRA同行（開始カットシーンあり）。
- 開始（カットシーン stage5-intro）：管理者の中枢に近い冷たい場所。TERRAが怯え（「ここ、怖い」）「ぜったいに勝てる？」と問う。RAYは答えられない（沈黙）。
- 開始（中央intro）：情景「世界を管理するものの、すぐ外」「ここを越えれば、管理者の中心だ」。
- 開始（内心 stageStart）：決意の入口「もう、迷わない。私は、この子を連れて進む」。
- ボス接近（内心 terraFound）：**転の予感「あの声が、テラだけを見ている。まさか——」**。
- ボス前（あの声）：**名指し＝回収**「お前がかばう、その個体。それが、最後の汚染源だ。引き渡せ」＋同じ光の匂わせ「お前の核は、私と同じ光。なのに、お前は壊れている」（選べるRAYを故障扱い）。
- RAYの決意（内心 eclipseReaction）：**過去→未来へ転換**「故障でもいい。最後の一人だから、私が守る。私が、そう決めた」。
- ボス撃破後：自分を受け入れる「この気持ちは、私のものだ。それでいい」。
- ※決意は科学者の遺言でなく、TERRA＋自分の選択。科学者は最後まで姿を見せない。

### Stage 6「支配中枢」（結・決着）🎯
- 役割：論理 vs 選択（心）の決着。**苦い勝利**で幕。TERRA は安全な場所で待機（離脱→再合流）。
- 開始（カットシーン stage6-awakening）：冒頭で**テラ離脱の一拍**「ここで待ってる。レイ、必ず戻ってきて」→RAY「テラを、安全な場所に残した。ここから先は、一人だ」。支配中枢の最奥で脈打つ核がRAYに反応し「最後の光」を渡す＝**攻撃強化を獲得する覚醒演出**。「私と同じ何か（同じ光）」は匂わせるだけ（謎は残す・科学者は出さない）。
  - ※RAYの攻撃強化はこのStage6開始の覚醒で獲得する（Stage5撃破後ではない）。強化はStage6の間のみ有効。
- 開始（中央intro）：情景・場所「一番奥の部屋」「終わりにする」。
- 開始（内心 stageStart）：選んだ道「テラのために。これは、私が選んだ道だ」。
- ボス前（あの声）：**自己定義＝決着**「私は管理者。この星を管理する者だ」「お前は人間をかばう。それは、星への裏切りだ」。
- RAY内心（対決前 eclipseReaction）：論破せず行動で答える「言い返さない。守ることが、私の答えだ」。
- ボス撃破後：「……終わった」。
- エンディング（カットシーン stage6-ending）：**群衆を出さない**。管理解除→**待っていたTERRAと再合流**（「レイ！」）→自由な空のセリフ交換→苦い勝利。⚠ **落書き・崩れたバリケード等は描かない**（描画制約：画面に出せないものを語らない）。苦さは props でなく事実・口調で「この星は、まだ傷だらけだ。すぐには、元に戻らない」。TERRA「次は、何する？」→ RAY「次は、私たちが決める」。締め「終わりではなく、始まり」。
- (回収)：星を守ることと人間（TERRA）を生かすことの両立を、"これから二人で目指す"希望として置く。達成でなく始まり。
