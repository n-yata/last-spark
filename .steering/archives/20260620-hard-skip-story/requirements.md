# Requirements: Hard Mode Story Skip

## User Request

Hard mode should not show story content. It should be positioned as a mode for players who want a more challenging game-focused experience.

## Scope

- In hard mode, skip story cutscenes and story overlay text.
- Keep normal mode story behavior unchanged.
- Keep hard mode gameplay tuning unchanged.
- Preserve clear progression and save updates even when ending story is skipped.

## Out of Scope

- Rewriting story scripts.
- Changing difficulty combat balance.
- Changing title/options UI copy unless required by implementation.
