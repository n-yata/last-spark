# 要求内容

## 概要

ステージ4・5のボスが他ステージボスの「焼き直し（使い回し）」で戦闘体験がつまらないため、ストーリーとステージのテーマに沿った固有のメカニクスとビジュアルを持つボスとして新規作成する。

## 背景

調査の結果、以下の流用構造が判明した。

- **Stage 5（使者 / ENVOY）**: Stage 2 の飛行ボス `FlyingBoss` をパラメータだけ変えて生成しており、固有クラス・固有アクションがゼロ。コード共有率 95〜98% の「色違い」状態。最も深刻。
- **Stage 4（環境管理機 / PURIFIER）**: 接地ボス `Boss` を継承した `PurifierBoss` で `spray`（扇状毒霧）という固有アクションは持つが、ビジュアルは Stage 1 ボスのリグ（`'boss'`）を流用した placeholder で、敵としての個性が立っていない。

また `docs/story.md` に「Stage 4-5 のボスは物語との接続を②フェーズで見直す」と既に課題として明記されている。物語上の意味は確定済みのため、それに噛み合うメカニクス・ビジュアルを与える。

## 実装対象の機能

### 1. Stage 5 ボス「使者 / ENVOY」の新規作成（テーマ: 決意・選択）

ECLIPSE の「最後の伝令」。論理を突きつける高速の刺客。プレイヤーに「読み（選択）」を強いる戦闘体験で、RAY の決意とシンクロさせる。

- 固有クラス `EnvoyBoss extends FlyingBoss` を新設（流用ゼロへ）
- 新アクション `blink`（逆サイドへ瞬間移動して挟む・残像エフェクト）
- 新アクション `lance`（任意角度の高速槍弾を時間差で複数発射）
- 専用ビジュアルリグ `bossEnvoy`（槍／矢じり型の鋭角シルエット）

### 2. Stage 4 ボス「環境管理機 / PURIFIER」の新規作成（テーマ: 揺らぎ・疑い）

「浄化という名の汚染」で星を殺す皮肉の象徴。プレイヤーの安全地帯をじわじわ奪い、「正しさが侵食される」感覚を体験させる。

- 新アクション `bloom`（プレイヤー足元に時限式の汚染床 HAZARD を能動的に増殖させる設置型攻撃）
- 既存 `spray` は phase2 で強化（2連射化）
- 専用ビジュアルリグ `bossPurifier`（汚染タンクを背負った接地機）

## 受け入れ条件

### Stage 5 ENVOY
- [ ] `EnvoyBoss` が独立したクラスとして存在し、`FlyingBoss` の直接生成ではなくなっている
- [ ] `blink`・`lance` が ENVOY 専用の行動抽選テーブルに閉じ込められ、他系統（FLYING/GROUND等）の抽選に混入しない
- [ ] phase1/phase2 で行動の重み・テンポが変化する
- [ ] 専用リグ `bossEnvoy` が表示され、`bossFlying` 流用ではなくなっている

### Stage 4 PURIFIER
- [ ] `bloom` でランタイムに汚染床が生成され、時限で消滅する
- [ ] phase2 で `bloom` の設置枚数・存続時間が増え、安全地帯が縮小する
- [ ] `bloom` が PURIFIER 専用の抽選テーブルに閉じ込められ、他系統に混入しない
- [ ] 専用リグ `bossPurifier` が表示され、`'boss'` 流用の placeholder ではなくなっている

### 共通
- [ ] HP 序列（使者26 < 浄化28 < 番人30 < コア40）を維持し、難易度カーブを崩さない
- [ ] `npm test` / `npm run lint` / `npm run typecheck` / `npm run build` が全て通る

## 成功指標

- Stage 4・5 のボスが、他ステージボスとは異なる固有のメカニクス（地形干渉 / 瞬間移動+狙撃）とビジュアルを持つ
- 各ボスの戦闘体験がステージの物語テーマ（揺らぎ / 決意）と噛み合っている

## スコープ外

以下はこのフェーズでは実装しません:

- Stage 1〜3・6 のボスの作り替え
- 新規 BGM・効果音の追加
- ストーリーテキスト（`src/config/story/`）の改訂（物語接続は確定済みのため）
- `lance` 弾の貫通挙動（初版は「高速だが非貫通」に留め、必要なら後段で検討）
