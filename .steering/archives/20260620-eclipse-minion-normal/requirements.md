# 要求内容

## 概要

ハードモードのエクリプス戦で召喚される雑魚敵の強さを normal 相当に固定する。
対象は HP とプレイヤーへのダメージ量で、エクリプス本体や道中 hard 雑魚の調整は維持する。

## 背景

前回調整でエクリプス本体の攻撃圧は下げたが、エクリプスが召喚する雑魚敵には hard のステージ係数とプレイヤー被ダメージ倍率が残っていた。
そのため、ボス本体よりも一緒に出る雑魚敵の強さが体感難度を押し上げている。

## 実装対象の機能

### 1. エクリプス召喚雑魚の HP normal 化
- CoreBoss が召喚する Enemy は hard 中でも normal の HP で生成する。

### 2. エクリプス召喚雑魚のダメージ normal 化
- 召喚 walker の接触ダメージを hard 被ダメージ倍率の対象外にする。
- 召喚 turret の弾ダメージも hard 被ダメージ倍率の対象外にする。

## 受け入れ条件

- [ ] エクリプス召喚雑魚は hard 中でも normal の HP になる。
- [ ] エクリプス召喚雑魚の接触ダメージは hard 中でも normal と同じになる。
- [ ] エクリプス召喚 turret の弾ダメージは hard 中でも normal と同じになる。
- [ ] 道中の hard 雑魚強化と hard 敵数増加は維持される。

## スコープ外

- エクリプス本体の追加弱体化
- 道中 hard 雑魚の弱体化
- 難易度 UI の変更
