# 要求内容

## 概要

Stage 5（ECLIPSE外縁部）を追加する。科学者の遺志を完全に受け取り、RAYの迷いが消える「決意」のステージ。ボスは「ECLIPSEの使者（高速型）」。ログのクライマックス（RAYへの直接的なメッセージ）が解錠される。

## 背景

ブロック1-2の基盤の上に Stage 5 のデータを載せる定型作業が中心。Stage 5 のログは「科学者の遺志＝物語の感情的クライマックス」であり、テキストの組み込みがとりわけ重要。高速型ボスは初の挙動なので、ヒット&アウェイの動きの実装がポイント。

## 実装対象の機能

### 1. Stage 5（ECLIPSE外縁部）ステージデータ
- 地形・敵・ログトリガー3本・ボストリガー
- 機械の密度が高まる外縁部の環境表現
- stage4 → stage5 の連結

### 2. ボス「ECLIPSEの使者（高速型）」
- スリムで流線型。高速移動・連続攻撃のヒット&アウェイ型
- 予測しにくい動き

### 3. Stage 5 ステージ開始演出（TERRA同行）
- `docs/story.md`「TERRAのセリフ > Stage 5」確定スクリプトの再生

### 4. Stage 5 確定テキスト組み込み
- 開始テキスト・ログ3本（RAYへの遺志）・ECLIPSEの語りかけ・RAYの内心
- 科学者の遺言ログ（「俺の息子は、もういない。でも、お前がいる」）の解錠

## 受け入れ条件

- [ ] Stage 5 に到達でき、ECLIPSEの使者ボスと戦える
- [ ] 高速移動・ヒット&アウェイの挙動が機能する
- [ ] ステージ開始演出で TERRA との確定スクリプトが再生される
- [ ] 開始テキスト・ログ3本・語りかけ・内心が確定版どおり表示される
- [ ] クリアで stage5 が clearedStages に記録される
- [ ] `npm test` / `npm run lint` / `npm run typecheck` / `npm run build` が通る

## 成功指標
- 科学者の遺志（RAYの存在意義）がログを通じて感情的に伝わり、決意の転換点として機能する
