# 要求: 各ボスに固有名を与え、ボスHPバーに正しく表示する

## 背景・課題

- ボスHPバー(`src/ui/BossHpBar.ts`)のラベルが `'まもりのきかい'` に**ハードコード**されており、
  全6ステージのボスが同じ名前で表示される(stage6 のラスボスまで「まもりのきかい」と出る)。
- story.md(北極星)には各ボスの**類型名**(守護機械・哨戒機 等)はあるが、固有名がない。
- シャビの指示: 「各ボスの表示されている名前が簡易すぎる。ちゃんと各ボスに名前をつけよう」。

## 決定事項(シャビ承認済み)

- 命名方針 = **固有コードネーム制**。世界観の核「光(RAY) vs 影(ECLIPSE)」に合わせ、
  ECLIPSE が配下に与えた**影・日食・不吉モチーフ**の固有名を当てる。

| Stage | 類型(従来) | 固有名(新規) | 由来 |
|-------|-----------|-------------|------|
| 1 | 守護機械(接地型) | ウンブラ (UMBRA) | 本影=最も濃い影。最初の番兵 |
| 2 | 哨戒機(飛行型) | コロナ (CORONA) | 日食時の光冠。空を巡る |
| 3 | 収容番人(重装型) | ヴェイル (VEIL) | 覆い隠す帳。収容・拘束 |
| 4 | 環境管理機(浄化型) | ミアズマ (MIASMA) | 瘴気=毒霧 |
| 5 | 使者(高速型) | ヘラルド (HERALD) | 伝令 |
| 6 | 管理AI本体(ラスボス) | ECLIPSE | 太陽を遮る影の核(既存名) |

## 受け入れ条件

1. 各ステージのボス戦で、HPバー上に**そのボス固有の名前**が表示される。
2. ボス名はステージデータ由来で、ステージ追加時に命名を強制できる(型で必須)。
3. story.md(北極星)のボス設定表に固有名が反映されている。
4. lint / typecheck / test / build が全て通る。
5. 各ボスが固有名を持つこと・HPバーがそれを表示することを検証するテストがある。
