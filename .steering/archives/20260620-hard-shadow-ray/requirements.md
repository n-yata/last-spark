# Requirements: Hard Mode Shadow RAY Boss

## User Request

In hard mode only, spawn a hidden boss immediately after defeating ECLIPSE. The hidden boss should have a shape similar to RAY, and its size should be the same as RAY.

## Scope

- Add a hard-mode-only post-ECLIPSE boss encounter in stage6.
- Keep normal mode behavior unchanged.
- The hidden boss uses RAY-sized dimensions and a RAY-like humanoid rig.
- Existing ECLIPSE summoned enemies should not carry into the hidden duel.
- Clearing the hidden boss should continue through the existing final clear flow.

## Out of Scope

- New story text or cutscene content.
- New raster assets.
- Changes to non-hard difficulty progression.
