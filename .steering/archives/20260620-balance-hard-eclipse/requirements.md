# 要求内容

## 概要

ハードモードの最終ボス「エクリプス」が強すぎるため、ラスボスの攻撃圧を少し緩和する。
同時に、ハードモードではボス前のエリア探索中に出現する雑魚敵の数を増やす。

## 背景

現状のハードモードは被ダメージ倍率と雑魚敵係数が乗るうえ、エクリプス本体も高い弾速・高ダメージ・短い行動間隔・多い召喚数を持っている。
その結果、最終戦の理不尽さが強く、道中探索の「ハードらしさ」は敵数ではなく敵個体の強化に偏っている。

## 実装対象の機能

### 1. エクリプス本体の難易度緩和
- stage6 のコアボス `ECLIPSE_CORE` の攻撃圧を少し下げる。
- 最終ボスとしての強さは維持しつつ、ハードでも回避・処理の余地を増やす。

### 2. ハードモード道中雑魚数の増加
- `SpawnSystem` が hard でステージ敵配置を増量する。
- normal の敵配置は既存のまま維持する。

## 受け入れ条件

### エクリプス本体の難易度緩和
- [ ] エクリプスの弾ダメージ・弾速・phase2 行動間隔・召喚密度が現状より緩和される。
- [ ] エクリプスは引き続き全ボス中の最終ボスとして成立する。

### ハードモード道中雑魚数の増加
- [ ] hard の `SpawnSystem` では pending される道中雑魚数が normal より増える。
- [ ] normal の `SpawnSystem` では敵数がステージ定義と一致する。
- [ ] 増量ロジックはユニットテストで検証される。

## 成功指標

- ハードのエクリプス戦で、雑魚処理とボス回避の両方に操作余地が残る。
- ハードの道中探索で、敵個体強化だけでなく出現数の増加も体感できる。

## スコープ外

以下はこのフェーズでは実装しません:

- 新しい敵種の追加
- ボス AI の新アクション追加
- 難易度 UI の変更
