# 敵の弱点表示削除 設計

## 方針

前回追加したシールドの内部耐久とダメージルールは残す。ただし、敵本体や HUD に追加で出していたシールド/弱点表示は削除し、プレイヤーには命中時の手応えとボス HP の減り方で差を感じさせる。

## 変更点

- `Boss` から発光シールド部位の GameObject と同期処理を削除する。
- `CoreBoss` のシールド表示同期呼び出しを削除する。
- `HUD.bossShieldRatio` と `BossHpBar` のシールドバー描画を削除する。
- `GameScene` / `UIScene` のシールド比率 registry 連携を削除する。

## 検証

- 関連するユニットテストを実行する。
- `npm test` と `npm run build` を実行する。
