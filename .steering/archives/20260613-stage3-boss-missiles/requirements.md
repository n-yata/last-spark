# 要求: stage3 ボスのギミック差別化(ミサイル攻撃)

## 背景・課題

stage3 のボス「収容番人(CONTAINMENT_WARDEN)」は、接地型の汎用 `Boss` を
`GROUND_WEIGHTS`(idle / move / shoot / jump)のまま流用しており、stage1 ボスと
**行動ギミックが完全に同一**(差分はパラメータのみ)。プレイ体感が stage1 と
変わらず、後半ステージとしての手応え・新鮮味が不足している。

## 要求

stage3 ボスに、stage1/stage2 と明確に異なる固有ギミックを追加して差別化する。
具体例として **ミサイル発射(放物線で着弾するアーティラリー攻撃)** を実装する。

### 受け入れ条件

1. stage3 ボスが、stage1 にない固有アクション「ミサイル」を持つ。
2. ミサイルは放物線(重力影響あり)を描き、プレイヤー周辺に着弾する。
   水平に飛ぶ stage1 の通常弾とは軌道・脅威が明確に異なる。
3. phase2 ではミサイルの本数・頻度が増し、攻勢が強まる。
4. 既存の stage1(接地)・stage2(飛行)ボスの挙動には影響しない。
5. ミサイル弾は見た目でも通常弾と区別できる(専用テクスチャ)。
6. 既存テストが壊れず、新ロジックに対する単体テストを追加する。
