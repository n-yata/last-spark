# ハードモード追加 設計

## 方針

難易度は `GameSettings` の一部として `difficulty: 'normal' | 'hard'` を保持する。既存セーブは読み込み時に `normal` を補完し、保存バージョンを更新する。

## UI

- 共通オプションメニューの音量設定パネルに `MODE: NORMAL/HARD` のトグルを追加する。
- タイトル画面とポーズ画面は既に同じ `createOptionsMenu` を使っているため、1 箇所の変更で両方から選べる。

## ゲームバランス

hard は以下で難しくする。

- プレイヤー被ダメージ倍率を上げる。
- 雑魚敵 HP 係数を上げる。
- walker 速度係数を上げる。
- turret 発射間隔係数を下げる。

ステージ別チューニングは既存の `StageTuning` に `enemyHpFactor` を追加し、難易度適用は純粋関数で行う。

## 注意点

- オプション変更中の既存出現済み敵へは即時反映しない。新規開始・リトライ・次ステージから反映される。
- 音量処理は `difficulty` を無視して従来どおり動作する。
