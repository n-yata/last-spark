# Requirements: Overall Polish

## Goal

LAST SPARK の既存体験を壊さず、ゲーム全体の見た目・手触り・読みやすさを底上げする。

## Scope

- Phaser で描画される HUD / 操作フィードバック / 戦闘演出 / 背景の軽量改善
- 既存の世界観「暗い廃墟基調 + 発光アクセント」を維持する
- 新ステージ、新ボス、新規ストーリー追加は行わない
- 外部 API、外部サービス、外部素材の追加は行わない

## Acceptance Criteria

- HUD がゲーム画面上でより読みやすく、状態変化が分かりやすい
- チャージ完了、被弾、ボス戦開始などの重要な瞬間に視覚フィードバックが増える
- 背景や空気感が単調になりすぎず、既存ステージの雰囲気を補強する
- 既存の unit / integration / build が通る
- 変更にハードコードされた URL、キー、シークレット、AWS 情報を含めない
