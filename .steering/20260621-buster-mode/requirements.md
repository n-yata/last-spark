# 要求内容

## 概要

ステージ6でのみ得られる「強化されたバスター」(RAYの攻撃強化 = 通常弾2発化 + チャージのビーム化)を、
全ステージでいつでも撃てるようにする新オプション「バスターモード」を追加する。難易度(ノーマル/ハード)
とは独立したトグルとし、どちらの難易度でも ON/OFF を選択できる。

## 背景

現状、RAYの攻撃強化(`empowered`)はステージ6(支配中枢)の静的属性として `GameScene.createPlayer` で
`stageId === 'stage6'` のときのみ付与される。プレイヤーが強化バスターの爽快感を全ステージで味わいたい、
あるいはやり込み/練習用に常時強化したいという要望に応えるため、強化を全ステージへ拡張する任意モードを
用意する。難易度設定とは独立させることで、ノーマル・ハードどちらのバランスでも強化を楽しめるようにする。

## 実装対象の機能

### 1. バスターモード設定の永続化

- `GameSettings` に `busterMode: boolean` を追加(既定 `false`)。
- localStorage のセーブ構造バージョンを v4 → v5 に上げ、既存セーブ(v4)から `busterMode: false` を補完して
  マイグレーションする(クリア進捗を失わない)。
- ユーザーは設定を変更でき、再起動後も保持される。

### 2. オプションメニューでのバスターモード切り替え

- タイトル/ポーズ共通のオプションメニュー(`optionsMenu.ts`)のルートに `BUSTER: ON/OFF` トグルを追加。
- 既存の `MUTE` トグルと同様、タップで即時切り替え + 即時永続化。
- 難易度(`MODE: NORMAL/HARD`)とは独立した別項目として並べる。

### 3. 全ステージでの強化バスター適用

- バスターモードが ON のとき、全ステージで RAY が `empowered` になる(通常弾2発化 + チャージビーム)。
- ステージ6は従来どおりバスターモードの ON/OFF に関係なく常に `empowered`(物語上の固有属性を維持)。

## 受け入れ条件

### バスターモード設定の永続化

- [ ] `GameSettings.busterMode` が型に追加され、既定値が `false`。
- [ ] 既存の v4 セーブ(busterMode フィールドなし)が `busterMode: false` を補完して v5 へ移行され、
      クリア進捗・ベストタイム・難易度設定が失われない。
- [ ] busterMode を ON にして再ロードしても ON が保持される。
- [ ] 不正な busterMode 値(boolean 以外)を含むセーブは既定値へフォールバックする。

### オプションメニューでのバスターモード切り替え

- [ ] オプションメニューのルートに `BUSTER: ON/OFF` が表示される。
- [ ] タップで ON ↔ OFF が切り替わり、ラベルが即時更新される。
- [ ] 切り替えが localStorage へ即時保存される。

### 全ステージでの強化バスター適用

- [ ] バスターモード ON のとき、ステージ1〜5でも通常弾が2発に分離し、チャージでビームが撃てる。
- [ ] バスターモード OFF のとき、ステージ1〜5は従来どおり非強化。
- [ ] ステージ6はバスターモードの ON/OFF に関わらず常に強化される。
- [ ] 強化適用判定が純粋関数として単体テストできる。

## 成功指標

- 既存テスト(SaveManager 等)が全て通る + 追加テストが通る。
- lint / typecheck / build がすべて成功する。
- 既存プレイヤーのセーブ進捗が移行後も保持される。

## スコープ外

以下はこのフェーズでは実装しません:

- バスターモードのアンロック条件(最初から選択可能とする)。
- 強化バスターの威力・見た目バランスの再調整(既存 `empowered` の挙動をそのまま流用)。
- 第3の難易度としての追加(あくまで難易度と直交するトグル)。
- バスターモード ON 時のスコア/ランキング上の区別。
