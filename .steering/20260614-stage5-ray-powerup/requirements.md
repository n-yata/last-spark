# 要求内容

## 概要

Stage5（外縁部）のボスを撃破したとき、クリア演出の中でRAYが「攻撃強化」を獲得し、続くStage6（支配中枢）でRAYの攻撃がパワーアップした状態でプレイできるようにする。

## 背景

物語のクライマックス（Stage6＝管理AIとの最終決戦）の直前に、プレイヤーへ「強くなった」という手応えと高揚を与えたい。Stage5の役割は story.md 上「決意（命令→意志の完成）」であり、その決意の到達点として攻撃強化を演出的に重ねることで、最終ステージへの盛り上がりを作る。

ただし強化の「きっかけ」の描き方は北極星 `docs/story.md` の方針と衝突しうるため、監督判断で以下を確定した（詳細は design.md「強化の源の演出」）：

- **科学者ログ/メモリは使わない**（story.md で全廃済み。科学者は語らせず匂わせるだけ、を厳守）。
- 強化の源は **案A「休眠コアとの共鳴」**：外縁部最奥で朽ちた古い機械（休眠コア）がRAYに反応し、最後の光を渡す。RAYは喋らず内心テキストのみ。「同類の気配」を説明せず匂わせる。

## 実装対象の機能

### 1. RAYの攻撃強化（パワーアップ）

- **通常攻撃**: 強化前は前方1発。強化後は弾が**上下2発**に分かれて出る（緩い斜め上下 ±18°）。
- **チャージ攻撃**: 強化前はチャージ弾1発。強化後は**持続レーザー（ビーム）**が出る。
  - ビーム発動中も**移動・ジャンプ可能**。
  - ビームの総威力は**チャージ弾と同等（約3）**。

### 2. 強化の獲得イベント（Stage5クリア演出）

- Stage5のボス撃破後、クリア演出（カットシーン `stage5-awakening`）の中で**自動的に**強化を獲得する。
- 演出は「休眠コアとの共鳴」（案A）。RAYの内心テキストのみ。**科学者ログは出さない**。

### 3. 強化状態の保持と適用範囲

- 強化が有効なのは **Stage6のみ**（Stage5クリア→Stage6への一時的な引き継ぎ）。
- **セーブには保存しない**（Phaser registry の揮発フラグで保持）。
- **リトライ維持**: Stage5を正規クリアしたプレイでは、Stage6で何度ゲームオーバー→リトライしても強化を維持する。
- **単体選択は素**: タイトルからStage6を単体選択／直接開始した場合は、強化なしの素のRAY。
- 全クリア（エンディング）後はフラグをクリアし、次のプレイに漏れない。

## 受け入れ条件

### RAYの攻撃強化
- [ ] 強化状態で通常攻撃すると、弾が上下2発（緩い斜め）で発射される。
- [ ] 強化状態でチャージ攻撃すると、持続レーザー（ビーム）が発動する。
- [ ] ビーム発動中もRAYは移動・ジャンプできる。
- [ ] ビームは敵・ボスの両方にダメージを与え、総威力はチャージ弾と同等（約3）に収まる。
- [ ] 非強化時は従来どおり通常1発／チャージ弾1発（既存挙動の回帰なし）。

### 強化の獲得イベント
- [ ] Stage5ボス撃破後、`stage5-awakening` カットシーンが再生される。
- [ ] カットシーンに科学者ログ/メモリの語りが**含まれない**（RAY内心と休眠コアの共鳴のみ、story.md 準拠）。
- [ ] カットシーン完了後、強化フラグが付与されStage6（ClearScene経由）へ進む。

### 強化状態の保持
- [ ] Stage5正規クリア→Stage6で、通常2発＋ビームが出る。
- [ ] Stage6でゲームオーバー→リトライしても強化が維持される。
- [ ] タイトルからStage6を単体選択した場合は素のRAY（強化が漏れない）。
- [ ] 全クリア→タイトル→Stage6単体選択でも素のRAY（フラグ残骸が漏れない）。

## 成功指標

- Stage6突入時に「強くなった」体感がある（通常2発の手数・ビームの一閃）。
- 既存の弾／戦闘ロジックに回帰バグがない。
- 複数アスペクト比（モバイル含む）でビームの当たり判定・射程が一定。

## スコープ外

以下はこのフェーズでは実装しません:

- 強化状態のセーブ永続化（再起動後も保持する仕組み）。
- Stage5以外での強化獲得、段階的（複数段）パワーアップ。
- 強化前後のHP・移動速度など攻撃以外のステータス変更。
- ビーム専用の高度な描画（多層グロー・パーティクル等）は最小限（単層発光＋フェードで可）。
- 休眠コアの専用背景SVG（フォールバック演出で可、後追い可能）。
