# 要求内容

## 概要

Stage 6（ECLIPSE支配中枢）とエンディングを追加し、全6ステージを完結させる。ラスボス「ECLIPSE本体」との複数フェーズ戦と、苦い勝利を描くエンディング演出を実装する。

## 背景

ブロック1-2の基盤と Stage 3-5 が完成している前提。Stage 6 は物語の決着であり、(1)複数フェーズのラスボス、(2)エンディング演出（人間が初めて直接描写される転換）という2つの特別要素を持つ。全クリアの SaveData 処理とタイトルへの帰還も含む。

## 実装対象の機能

### 1. Stage 6（ECLIPSE支配中枢）ステージデータ
- 地形・敵・ログトリガー（Stage 6 はボス前後ログなし＝序盤ログのみ）・ボストリガー
- 支配中枢の環境表現
- stage5 → stage6 の連結（最終ステージ＝nextStageId なし）

### 2. ラスボス「ECLIPSE本体」（複数フェーズ）
- 人型でない巨大コア。`docs/story.md` の方向性に従う
- 前半: 配下を呼び出す支援型フェーズ
- 後半: コア直接攻撃型フェーズ
- ECLIPSEの語りかけ（「地球を守るために、お前を終了する」）はボス戦直前に発生

### 3. エンディング演出シーン
- `docs/story.md`「Stage 6 結末演出の詳細構成」に従う
- ECLIPSE撃破 → 管理解除テキスト → 演出シーン（廃墟の外・人間が初めて直接描写・争いの痕跡）→ TERRAとのセリフ交換 → エンディングテキスト
- 苦い勝利（楽園ではない・再生の始まり）のトーン

### 4. 全クリア処理
- stage6 クリアで全クリア状態を保存し、タイトルへ帰還
- タイトルでの全クリア表示（素地）

## 受け入れ条件

- [ ] Stage 6 に到達でき、ECLIPSE本体と戦える
- [ ] ボスが複数フェーズ（支援型→直接攻撃型）で挙動する
- [ ] ボス戦直前にECLIPSEの語りかけが表示される
- [ ] 撃破後にエンディング演出（管理解除→人間描写→TERRAセリフ→エンディングテキスト）が再生される
- [ ] 全クリアが保存され、タイトルへ戻る
- [ ] Stage 6 の開始テキスト・序盤ログ・内心が確定版どおり表示される
- [ ] `npm test` / `npm run lint` / `npm run typecheck` / `npm run build` が通る

## 成功指標
- 全6ステージを通しでクリアでき、苦い勝利のエンディングが感情的に着地する
- 「終わりではなく、始まり」というテーマがエンディングで伝わる

## スコープ外
- BGM 拡張 → ブロック6（並行可）
- 周回プレイ・難易度選択等の追加機能
