# 要求内容

## 概要

Stage 4（汚染地帯）を追加する。環境破壊の実態を目撃し、ECLIPSEの論理に揺らぐ物語の「揺らぎ」のステージ。ボスは「環境管理機（浄化型）」。

## 背景

ブロック1（テキスト基盤）・ブロック2（演出シーン・SaveData・ボス後フロー）が完了している前提で、Stage 4 はそれらの仕組みに「データと確定テキスト」を載せる定型作業が中心となる。TERRA 同行後の最初のステージで、ステージ開始演出に TERRA が登場する。

## 実装対象の機能

### 1. Stage 4（汚染地帯）ステージデータ
- 地形・敵・ログトリガー3本・ボストリガー
- 環境破壊を感じさせる環境ストーリーテリング要素（汚染された大地・淀んだ空気の表現）
- stage3 → stage4 の連結

### 2. ボス「環境管理機（浄化型）」
- 浄化タンク搭載の機械。毒・スプレー系の範囲攻撃
- 「浄化」という名の汚染をまき散らす皮肉な攻撃表現

### 3. Stage 4 ステージ開始演出（TERRA同行）
- `docs/story.md`「TERRAのセリフ > Stage 4」確定スクリプトの再生

### 4. Stage 4 確定テキスト組み込み
- 開始テキスト・ログ3本（環境への怒りと悲しみ）・ECLIPSEの語りかけ・RAYの内心

## 受け入れ条件

- [ ] Stage 4 に到達でき、環境管理機ボスと戦える
- [ ] 毒・スプレー系の範囲攻撃が機能する
- [ ] ステージ開始演出で TERRA との確定スクリプトが再生される
- [ ] 開始テキスト・ログ3本・語りかけ・内心が確定版どおり表示される
- [ ] クリアで stage4 が clearedStages に記録される
- [ ] `npm test` / `npm run lint` / `npm run typecheck` / `npm run build` が通る

## 成功指標
- 「人類は確かに地球を壊した」という揺らぎがテキストと環境から伝わる
