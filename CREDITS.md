# CREDITS / ライセンス出典

`LAST SPARK` で使用しているアセットとその出典・ライセンスを記録します。
第三者の知的財産(キャラクター・スプライト・名称・音楽)は使用していません。

## グラフィック

- **すべて自作(コード生成)**: プレイヤー・敵・ボス・弾・地形・UI のグラフィックは、
  Phaser の図形描画 API を用いて実行時に生成したオリジナルのプレースホルダです。
  - 実装: `src/scenes/PreloadScene.ts`
  - PWA アイコン: `scripts/generate-icons.mjs`(コード生成、`public/icons/`)
- 外部から取得した画像素材は使用していません。

## サウンド / BGM

- MVP では音声素材を同梱していません。
- 将来素材を追加する場合は、CC0 等の商用利用可能ライセンスの素材に限定し、
  本ファイルに出典・ライセンスを追記します。

## フォント

- ゲーム内テキストはブラウザ標準の monospace フォント(システムフォント)を使用しています。
  独自フォントファイルは同梱していません。

## ライブラリ

各 OSS ライブラリのライセンスは `node_modules` 内の各パッケージおよび `package.json` を参照してください
(Phaser: MIT, Vite: MIT 等)。

---

> 方針: 既存作品(ロックマン等)の素材・名称・固有名は一切使用しません。
> キャラ・世界観・名称はすべて本プロジェクトのオリジナルです。
